# Typography

---

## Font family

Snowball's design took the side of using a close source and strongly marked font
called [PolySans](https://wearegradient.net/polysans/) created by
[Gradient](https://wearegradient.net/about/), an independant type foundry.

We use the `Standard` and `Wide` versions of this font in `400`, `600` and `800`
in both `normal` and `italic` styles.

> ##### ℹ️ Info
>
> ---
>
> When you download the open-source packages from Snowball's repository like
> [Fractal](https://github.com/snowball-tech/fractal) or
> [Glacier](https://github.com/snowball-tech/glacier), this font is
> _(obviously)_ not packaged.  
> If you want to use a close fallback, you can use:
>
> ```css
> .poly-sans-fallback {
>   font-family: system-ui, Arial, sans-serif;
>   letter-spacing: 0.2px;
> }
> .poly-sans-wide-fallback {
>   font-family: 'Arial Black', system-ui, sans-serif;
>   letter-spacing: 0.2px;
> }
> .poly-sans-fallback h1,
> h2,
> h3,
> h4,
> h5,
> h6,
> .poly-sans-wide-fallback h1,
> h2,
> h3,
> h4,
> h5,
> h6 {
>   letter-spacing: 0;
> }
> ```

For email content and if you can't use PolySans, use "system-ui" and then
"Arial" as fallbacks.

> ##### ℹ️ Info
>
> ---
>
> When using the "system-ui"; "Arial" or "Arial Black" fallback (either for emails or because
> you don't have access to the PolySans font), you should also use a `0.2px`
> letter spacing to improve readability, except for headings.

```css
.email {
  font-family: system-ui, Arial, sans-serif;
  letter-spacing: 0.2px;
}
.email-wide {
  font-family: system-ui, 'Arial Black', sans-serif;
  letter-spacing: 0.2px;
}
.email h1,
h2,
h3,
h4,
h5,
h6,
.email-wide h1,
h2,
h3,
h4,
h5,
h6 {
  letter-spacing: 0;
}
```

## Levels

You can use 4 levels of typography in Fractal:

- **Display** are used for desktop and mobile marketing headings and punchlines.  
  It is subdivided in 2 levels, corresponding to each `<hX />` element.  
  There is also a separate `wide` variation.
- **Heading** are used for desktop and mobile product headings, titles and
  punchlines.  
  It is subdivided in 4 levels, corresponding to each `<hX />` element.  
  Levels 3 and 4 are also available in `link` variations
- **Body** are used for desktop and mobile standard paragraphs and contents.  
  It is subdivided in 2 levels _(regular and low)_.  
  Each is available in "normal", `median`, `bold` and `link` variations.
- **Caption** are used for desktop and mobile captions and small texts.
  It comes in 3 variations: `median`, `bold` and `link`.

### Display

**Font:** `PolySans, system-ui, Arial, sans-serif`  
**Weight:** `800`  
**Line height:** `font-size * 1.4`

#### Desktop

**Sizes:** `80px` (`display-1`), `72px` (`display-2`)

<Typeset
  fontFamily={`PolySans, system-ui, Arial, sans-serif`}
  fontSizes={[80, 72]}
  fontWeight={800}
  sampleText="May the force be with you!"
/>

#### Mobile

**Sizes:** `48px` (`display-1`), `40px` (`display-2`)

<Typeset
  fontFamily={`PolySans, system-ui, Arial, sans-serif`}
  fontSizes={[48, 40]}
  fontWeight={800}
  sampleText="May the force be with you!"
/>

#### Email and fallback

**Font:** `system-ui, Arial, sans-serif`  
**Letter spacing:** `0px`

<Typeset
  fontFamily={`system-ui, Arial, sans-serif`}
  fontSizes={[80, 72, 48, 40]}
  fontWeight={800}
  sampleText="May the force be with you!"
/>
